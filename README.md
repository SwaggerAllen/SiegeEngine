# SiegeEngine

A Claude Code plugin + MCP server + git-backed state model for
drafting, reviewing, and approving a project's architecture in
compressed tiers.

The code, install instructions, and full documentation live under
**[siege-engine/](siege-engine/)** — see
**[siege-engine/README.md](siege-engine/README.md)**.

Deploy notes: [siege-engine/DEPLOYMENT.md](siege-engine/DEPLOYMENT.md).

User-facing reference (workflows + slash commands):
[siege-engine/frontend/src/content/cheatsheet.md](siege-engine/frontend/src/content/cheatsheet.md),
served in-browser at `https://siege.strutco.io/cheatsheet`.
